You have to be on mobile in order to see it, try resizing the browser window.

    <span class="cart-icon">
        <a class="vendor-cart-icon">
            <svg>....</svg>
            <span class="products-count">10</span>
        </a>
    </span>
    <span>View cart</span>

    <span class="calcul-animation">
        <span class="dot"></span>
        <span class="dot"></span>

</button>
