# Logo

Logo presented on header. It sends user to homepage when clicked. Can be applied by .logo css class

 





<div class="logo">
    <a href="#logo">
        <svg>....</svg>
    </a>
</div>

for Foodpanda
 






<div class="logo">
    <a href="#logo">
        <svg>....</svg>
        <span class="company-name">for Foodpanda</span>
    </a>
</div>
